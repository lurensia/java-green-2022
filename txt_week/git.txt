

1.git초기화
git init

2.사용자 생성
git config --global user.name "lusia"
git config --global user.email "woder318@gmail.com"


3.스냅샷(.은 현재 폴더내의 전체 파일)
git add .

4.사진첩 보관
git commit -m "[1조 첫째주 내용 ]"

5.브랜치확인
git branch

6.브랜치 생성 및 이동
git branch dev

7. 브랜치 변경
2조:git checkout dev
1조:git checkout master
